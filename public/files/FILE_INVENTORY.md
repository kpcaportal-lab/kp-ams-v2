## 🔤 Typography System

### Fonts Used
- **Playfair Display** (Serif) - Headings and titles
  - Source: Google Fonts (free)
  - Weights: Bold 700
  
- **System Font / Roboto** (Sans-serif) - Body and UI
  - Source: Built-in or Google Fonts
  - Weights: Regular 400, Bold 700

### Text Sizes
- **H1**: 48px - Page titles
- **H2**: 36px - Section headers
- **H3**: 28px - Subsection headers
- **H4**: 20px - Card titles
- **Body**: 16px - Main content
- **Small**: 14px - Captions and labels

## 🖨️ Print Specifications

### PDF Export
1. Open index.html in browser
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Save as PDF or Print directly

### Print Quality
- 300 DPI recommended for print
- Full color recommended
- Margins: 0.5 inches recommended
- Paper: A4 or Letter size

### Print Tips
- Use color printer for best results
- Test print on sample page first
- Save PDF for archival
- Keep both digital and print versions
