### 1. Missing Input Validation on Create Endpoints
**Severity:** HIGH  
**Affected Routes:**
- `POST /api/proposals` — no schema validation
- `POST /api/assignments` — no schema validation
- `POST /api/invoices` — no schema validation
- `POST /api/tickets` — no schema validation

**Current:** Trusts `req.body` directly  
**Risk:** Malformed data, SQL injection via unescaped values  
**Fix:** Add Zod/Joi validation schemas

**Code Pattern:**
```typescript
// Add to each route:
import { z } from 'zod';

const CreateProposalSchema = z.object({
  client_id: z.string().uuid(),
  proposal_type: z.enum(['new', 'revision']),
  assignment_type: z.enum(['internal_audit', 'forensic', 'overseas', 'mcs', 'ifc']),
  quotation_amount: z.number().positive(),
  // ... other fields
});

router.post('/', async (req, res) => {
  try {
    const validated = CreateProposalSchema.parse(req.body);
    // ... use validated data
  } catch (err) {
    if (err instanceof z.ZodError) {
      return res.status(400).json({ error: 'Validation failed', details: err.errors });
    }
    throw err;
  }
});
```

```typescript
import rateLimit from 'express-rate-limit';

const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15 minutes
  max: 100, // limit each IP to 100 requests per windowMs
  message: 'Too many requests, please try again later.',
  standardHeaders: true, // Return rate limit info in headers

  windowMs: 15 * 60 * 1000,
  max: 5, // 5 login attempts per 15 mins

});

```

### 4. Missing Transaction Support for Multi-Step Operations
**Severity:** MEDIUM  
**Affected:** 
- `POST /api/assignments` — creates assignment + 12 fee allocations
- `POST /api/proposals/:id/generate-assignments` — creates multiple records

**Risk:** Partial failures leave inconsistent state  
**Fix:** Wrap in explicit transactions

```typescript
router.post('/', async (req, res) => {
  const client = await pool.connect();
  try {
    await client.query('BEGIN');
    
    const assignment = await client.query(`INSERT INTO ...`, [...]);
    
    const allocationPromises = Array.from({ length: 12 }, (_, i) =>
      client.query(`INSERT INTO fee_allocations ...`, [...])
    );
    await Promise.all(allocationPromises);
    
    await client.query('COMMIT');
    res.status(201).json(assignment.rows[0]);
  } catch (err) {
    await client.query('ROLLBACK');
    console.error(err);
    res.status(500).json({ error: 'Server error' });
  } finally {
    client.release();
  }
});
```
